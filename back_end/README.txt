=====================================================
README for Driveroo backend
=====================================================
1. Enter into the flask folder and open terminal
2. Input "python flask.py" in the terminal
3. Now, the server has started if you see: * Running on http://0.0.0.0:5001/ (Press CTRL+C to quit)
4. Go back to the upper folder, open another terminal
5. Input "python detection_main.py"
6. Now, the face and pupil detection system has ran successfully
